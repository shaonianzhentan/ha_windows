## Windows应用快捷键

- 语音控制 Voice：`Alt` + `V`
- 显示设备界面 Device：`Alt` + `D`
- 显示家的界面 Home：`Alt` + `H`
- 显示网页界面 Web：`Alt` + `W`
- 全屏显示：`F11`
- 【播放器】音量增加：`Alt` + `上键↑`
- 【播放器】音量减少：`Alt` + `下键↓`
- 【播放器】上一曲：`Alt` + `左键←`
- 【播放器】下一曲：`Alt` + `右键→`
